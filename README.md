vim_files
=========

customized vim config files

INSTALL INSTRUCTIONS
====================

download this repo:
mkdir ~/.vim
git clone https://github.com/horahh/vim_files ~/.vim

link  the .vimrc with the vimrc.txt

   ln -s ~/.vim/vimrc.txt ~/.vimrc

load scripts:

   cd .vim
   git submodule init
   git submodule update
